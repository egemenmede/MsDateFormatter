MsDateFormatter
=========

Microsoft Date formatının normal tarih formatına çevirmek için kullanışlı bir kütüphane.

Kullanım Şekli
=========

Örnek Sayfası: http://jsfiddle.net/DeliPenguen/hm63du8w/1/

msDateFormatter.convert("/Date(1413444063898)/", "/");
msDateFormatter.convert("/Date(1413444063898)/", ".");
msDateFormatter.convert("/Date(1413444063898)/", "-");

Boyutu
=========
Yalnızca 855 Byte
